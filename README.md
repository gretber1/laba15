# # Фирма грузоперевозок  
Windows Forms приложение на C++/CLI

## Описание проекта
Данное приложение представляет собой Windows Forms программу, разработанную на языке **C++/CLI** в среде **Visual Studio**.  
Программа предназначена для управления тарифами фирмы грузоперевозок и реализует полный набор функций для работы с данными: добавление, редактирование, удаление, поиск, сортировку, а также сохранение и загрузку данных из файла.

Проект разработан в рамках учебного задания и демонстрирует использование объектно-ориентированного подхода, графического интерфейса и работы с файлами.

---

## Основной функционал
- Работа с несколькими окнами (главное окно и окно тарифов)
- Режим администратора с ограничением доступа к функциям
- Добавление, редактирование и удаление тарифов
- Поиск тарифа с минимальной итоговой стоимостью
- Сортировка данных по столбцам таблицы
- Проверка корректности вводимых данных
- Сохранение и загрузка данных в текстовый файл
- Защита от выхода без сохранения данных
- Подтверждение критических действий пользователя

---

## Окна приложения
### 1. Главное окно (`MainForm`)
- Включение / отключение режима администратора
- Переход к окну управления тарифами
- Выход из приложения

### 2. Окно тарифов (`TariffsForm`)
- Отображение списка тарифов в `DataGridView`
- Управление тарифами 
- Поиск минимального тарифа
- Сохранение и загрузка данных через диалоги выбора файлов
