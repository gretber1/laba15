# # Фирма грузоперевозок  
Windows Forms приложение на C++/CLI

## Описание проекта
Данное приложение представляет собой Windows Forms программу, разработанную на языке C++/CLI в среде Visual Studio.
Программа предназначена для управления тарифами фирмы грузоперевозок и реализует полный набор функций для работы с данными: добавление, редактирование, удаление, поиск, сортировку, а также сохранение и загрузку данных.

Проект разработан в рамках учебного задания и демонстрирует использование объектно-ориентированного подхода, графического интерфейса пользователя, работы с файлами и базой данных SQLite.

---

## Основной функционал

Работа с несколькими окнами (главное окно и окно тарифов)

Режим администратора с ограничением доступа к функциям

Добавление, редактирование и удаление тарифов

Поиск тарифа с минимальной итоговой стоимостью

Сортировка данных по столбцам таблицы

Проверка корректности вводимых данных

Хранение данных в базе данных SQLite

Сохранение данных из базы данных в текстовый файл

Загрузка данных из текстового файла в базу данных

Явное сохранение изменений в БД по кнопке (для администратора)

Защита от выхода без сохранения данных

Подтверждение критических действий пользователя

---

## Окна приложения
1. Главное окно (MainForm)

Включение / отключение режима администратора

Переход к окну управления тарифами

Выход из приложения

2. Окно тарифов (TariffsForm)

Отображение списка тарифов в DataGridView

Управление тарифами (добавление, редактирование, удаление)

Поиск минимального тарифа

Загрузка данных из базы данных

Сохранение данных в базу данных

Экспорт данных из базы данных в текстовый файл

Импорт данных из текстового файла в базу данных
---

## Запуск проекта

Открыть проект в Visual Studio

Убедиться, что файл базы данных SQLite находится в каталоге проекта

Собрать решение (Build)

Запустить приложение (Start)

Программа компилируется в исполняемый файл .exe
